# CSharpStudies

Este projeto é um repositório de estudos e experimentos em C#. O objetivo principal é reunir exemplos, exercícios e pequenas aplicações para aprimorar o conhecimento na linguagem C#, abordando desde conceitos básicos até tópicos mais avançados.

## Objetivos

- Consolidar o aprendizado em C# por meio de exercícios práticos.
- Explorar diferentes áreas e funcionalidades da linguagem.
- Servir como referência para consultas futuras.

## Estrutura do Projeto

O repositório está organizado em subdiretórios, cada um dedicado a um tema ou módulo de estudo. Exemplos de tópicos que podem ser abordados:

- Fundamentos do C# (variáveis, tipos de dados, operadores)
- Programação Orientada a Objetos
- Manipulação de arquivos
- Coleções e LINQ
- Tratamento de exceções
- Threads e programação assíncrona
- Interação com bancos de dados
- Criação de interfaces gráficas (WinForms, WPF)
- Testes unitários

## Como Executar os Exemplos

1. Certifique-se de ter o [.NET SDK](https://dotnet.microsoft.com/download) instalado em sua máquina.
2. Navegue até o diretório do exemplo desejado.
3. Execute os comandos:

   ```bash
   dotnet build
   dotnet run
   ```
